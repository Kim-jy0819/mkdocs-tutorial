# command
`pip install mkdocs-material mkdocstrings[python] mkdocs-section-index mkdocs-gen-files mkdocs-literate-nav`<br>
`mkdocs build` site 디렉토리를 만듭니다.<br> 
`mkdocs serve` build 하지 않고 바로 server를 띄울수도 있습니다.<br>
